== Description ==

This is a BuddyPress plugin that allows you to select and display featured posts sitewide.
Basically, it adds a checkbox "Add this post/page to Featured Posts" when adding/editing posts or pages.


== Installation ==

= WordPress 2.9 and above = 

* Copy the directory buddypress-featured-posts to wp-content/plugins.
* Activate Sitewide.

== Usage ==

* When editing a post/page, check or uncheck "Add this post/page to Featured Posts" at the bottom of the page.
